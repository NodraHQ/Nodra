01 — Overview

Final Project

The Final Project is the Academy's capstone: instead of a new topic, it asks you to apply what Modules 01 through 09 covered — blockchain fundamentals, wallets, DeFi mechanics, and security — to one connected, realistic scenario.

This module will outline the project's structure, what will be evaluated, and how to submit your work for community review once the full module is published.

02 — What You'll Learn

What You'll Learn

  • How the capstone project is structured, module by module.
  • What deliverables will be expected by the end of the project.
  • How the project will be evaluated, and against what criteria.
  • How to set up a testnet environment to complete the project safely.
  • How peer review will work once the full module is published.
03 — Prerequisites

Prerequisites

  • Completed Module 01 — Blockchain.
  • Completed Module 02 — Wallets.
  • Familiarity with the DeFi track (Modules 05–08) is recommended.
  • Completed Module 09 — Security.
04 — Glossary

Glossary

Capstone
A final project designed to apply everything learned across a full course track.
Portfolio
A collection of completed projects used to demonstrate practical skill.
Peer Review
A process where fellow learners evaluate and give feedback on submitted work.
Deployment
The act of publishing a smart contract or application to a live network.
Testnet
A blockchain network used for testing, where assets have no real value.
